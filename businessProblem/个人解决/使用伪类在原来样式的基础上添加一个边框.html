使用伪类在原来样式的基础上添加一个边框.

小安小于 1 分钟

.towns-party {
  width: 25%;
  height: 56px;
  border-radius: 1px;
  background: #faebec;
  position: absolute;
  top: 10%;
  left: 40%;
  border: 1px solid #b53237;
  text-align: center;
  line-height: 56px;
  font-size: 20px;
  letter-spacing: 16px;
  font-family: PingFangSC-Medium, PingFang SC;
  font-weight: 500;
  color: #8b2024;
  // position: relative;
  &:active::after {
    content: '';     //content 属性与 :before 及 :after 伪元素配合使用，来插入生成内容。
    width: 110%;   //  制作一个超出原本的边框
    height: 72px;   //制作一个超出原本的边框
    position: absolute;
    top: -17%;
    left: -5%;
    border: 1px solid #f5cfd2;
  }
}
